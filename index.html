Lenguajes de programación

Html

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear páginas web. Define la estructura y el contenido de una página mediante etiquetas () que organizan elementos como texto, imágenes, enlaces, formularios y otros componentes multimedia.

css

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo que se usa para controlar la apariencia y el diseño de las páginas web escritas en HTML. Permite aplicar colores, fuentes, márgenes, tamaños y posicionamiento de los elementos para mejorar la presentación de un sitio web.

JavaScript

JavaScript (JS) es un lenguaje de programación que se usa para hacer que las páginas web sean interactivas y dinámicas. Mientras que HTML estructura el contenido y CSS lo diseña, JavaScript agrega funcionalidades como animaciones, validaciones de formularios y actualizaciones en tiempo real sin recargar la página.
